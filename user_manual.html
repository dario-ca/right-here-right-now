User manual

How to use this powerful interactive tool

Map

Click and drag to move around the map.
Use the controls on the bottom left to zoom in / out and switch between areal view and classic map.

Selections

Nearby

Click on the Nearby button to select a rectangle around your current location
Remember to configure your browser to allow this website to access to your location.

Rectangle

To select a rectangle on the map click on the Rectangle button. Then, click on the map and drag while the button is still pressed.
Release the mouse button to confirm your selection.

Path

To select a path on the map click on the Path button. Then, click on the map to select the starting point of your path. Every further click on the map will constraint the path to pass through that position. The last click on the map is the destination.
Click on the Path button again to confirm.

Layers

Selection

The panel on the left of the application window let the user to customize the visible layers on the map
Click on the checkbox to select / deselect the whole group. Click on a single layer to enable or disable it.

Time range

The Security and Information layers group can show data about the last week or the last month.
Click on the Week or Month button to switch between the two modalities.

Security

The security layer group displays information about the crimes
  • Personal Assault: crimes which directly threat the physical security of the user
  • Property Crime: crimes which damage the user economically
  • Unsafe Area: crimes that doesn't threat the user's security directly but they mark the neighborhood as a "bad area"
  • Minor Crime: all the other minor crimes which provoke small or none damage to the user, such as gambling

Information

  • Pothole: a depression or hollow in a road surface
  • Light: broken streetlight. If more light than 2 lights are broken together the icon on the map will be flagged with a warning
  • Abandoned vehicle: a vehicle which has been reported to be not used for at least a week or in very bad conditions

Mobility

  • Divvy: show the position of the Divvy's stations
  • Bus: crimes which damage the user economically

Social:

  • Twitter: crimes which directly threat the physical security of the user

Point of interest:

  • Restaurant: crimes which directly threat the physical security of the user
  • Bar: crimes which directly threat the physical security of the user
  • Important Place: crimes which directly threat the physical security of the user
